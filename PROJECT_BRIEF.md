# PROJECT_BRIEF

## Цел
Статичен сайт за подготовка за ДЗИ в 12. клас с две основни библиотеки:
- литература
- български език

## Решения
- Astro статичен build
- JSON content collections
- мобилен first, edge-to-edge усещане
- автоматичен dark/light режим според системата
- външни линкове за четене на произведенията
- редакционна визуална посока с еднаква дизайн система за layout, catalog и detail страници
- лек motion слой с Astro ClientRouter и CSS анимации с respect към prefers-reduced-motion

## Ограничения
- без backend
- без тежки медии
- Cloudflare Pages единичен asset под 25 MiB
